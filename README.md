# 妙覚寺 福岡窓口 - ランディングページ

佐賀に600年続く「竹原山 妙覚寺」の公式出張所、妙覚寺 福岡窓口のランディングページです。

## 🎯 プロジェクト概要

- **目的**: 檀家義務なし、お布施定額制の読経・葬儀・法事サービスのLP
- **デザイン**: 和風モダン、洗練された和紙テクスチャ、金色アクセント
- **技術スタック**: Hono + TypeScript + Cloudflare Pages

## ✨ 実装済み機能

### 1. **洗練された和風デザイン**
- ローディングスクリーン
- 固定ヘッダー（半透明・ブラー効果）
- 家族写真を配置したヒーローセクション
- SVGアイコンを用いた3つの理由セクション
- レスポンシブ対応

### 2. **料金プラン表示**
- 4つの基本プラン（火葬式・標準・安心・法要）
- 推奨プランバッジ表示
- 仲介料ゼロの明確な価格表示

### 3. **🎨 モーダル式BTOカスタマイザー（NEW）**
- **モーダル（ポップアップ）表示**: 「もっと細かくカスタマイズ」ボタンで集中できる見積もり画面を表示
- **インタラクティブなプラン選択**: セレクトボックスで基本プラン（火葬式/標準/安心/法要）を選択
- **オプション追加**: 戒名ランクアップ、追加法要、供養サービス、遠方出張費など8種類のオプション
- **リアルタイム見積もり**: 選択内容に応じて合計金額を自動計算（右側スティッキーサマリー）
- **LINE連携**: カスタマイズした見積もり内容をLINEで送信可能
- **カテゴリー分類**: 戒名/法要/供養/出張の4カテゴリー

### 4. **🏛️ プレミアムブランディング（NEW）**
- **源流の重み**: 「應永三十一年創建 ─ 九州日蓮宗 源流之寺」をヒーローに配置
- **二重の責任構造**: 宗教的責任（妙覚寺）と社会的責任（合同会社 海紹陽）を明文化
- **副住職プロフィール**: 山田紹智のプロフィールセクションで「顔の見える安心感」を提供
- **価格の納得感**: 寺院直販による中間マージンゼロの適正価格を説明

### 5. **ご依頼の流れ**
- 3ステップのわかりやすい説明
- 大きな番号表示で視認性向上

### 6. **SEO最適化**
- メタタグ完備
- OGタグ設定
- 構造化データ（今後実装予定）

## 📋 次のステップ

### 本番環境へのデプロイ

1. **Cloudflare API Key設定**
   - Deployタブで Cloudflare API Key を設定

2. **Cloudflare Pagesへデプロイ**
   ```bash
   npm run build
   npx wrangler pages deploy dist --project-name myokakuji-lp
   ```

### GitHub連携

1. **GitHub認証設定**
   - GitHubタブで認証を完了

2. **リポジトリへプッシュ**
   ```bash
   git remote add origin https://github.com/kaisyoyo1-blip/data1.git
   git push -u origin main
   ```
   ✅ **完了しました！**

### カスタマイズ

1. **連絡先情報の更新**
   - 電話番号: `tel:0123456789` → 実際の番号に変更
   - LINE URL: `#` → 実際のLINE URLに変更

2. **住所情報の確認**
   - 佐賀本院: 佐賀県佐賀市伊勢町8番12号
   - 福岡窓口の住所を追記
